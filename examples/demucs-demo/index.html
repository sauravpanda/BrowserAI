Demucs — In-Browser Source Separation

Split a song into drums, bass, other, and vocals, running fully client-side with BrowserAI + onnxruntime-web.

Uses smank/htdemucs-onnx via onnxruntime-web. Stick with the WASM execution provider — ort-web 1.21's WebGPU backend has a state bug that corrupts output on subsequent inference calls for this model. Run the demo via npx serve . from the repo root so the bundled serve.json sends the COOP/COEP headers that unlock multi-threaded WASM (otherwise ORT runs on 1 thread and it's ~4× slower).
